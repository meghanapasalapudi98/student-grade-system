# 🎓 Student Grade System

A REST API built with Java Spring Boot.

## Features
- Add students
- View all students
- H2 in-memory database
- Spring Data JPA

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## API Endpoints
| Method | URL | Description |
|--------|-----|-------------|
| GET | /students | Get all students |
| POST | /students | Add a student |

## How to Run
1. Clone this repo
2. Run `.\mvnw spring-boot:run`
3. Open `http://localhost:8081/students`
